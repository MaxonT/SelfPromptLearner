# SoulPrint: 终极心理学镜像系统升级方案

**当前完成度：80%**
(已完成：六大模型关键词库、16情绪核心映射、基础雷达图与3D星云、角色卡片框架。
**缺失部分**：心理学模型深度科普模块、维基百科权威链接、更细致的口诀记忆辅助、以及界面上的权威背书强化。)

## 核心目标
将 SoulPrint 从一个单纯的数据可视化工具，升级为**权威的心理学教育与自我探索平台**。不仅展示“你是什么”，还要告诉用户“为什么是这样”以及“背后的科学依据”。

## 升级内容详解

### 1. 权威背书与科普模块 (Psychology Glossary)
在 `SoulPrint` Tab 下方新增 "📚 Theoretical Basis" (理论基石) 区域。
*   **六大模型详解**：为 Big Five, MBTI, Enneagram, Jungian, DISC, HEXACO 每个模型建立专属折叠卡片 (Expander)。
*   **内容包含**：
    *   **定义**：引用权威定义 (如 APA 心理学词典)。
    *   **维度解释**：详细解释每个维度的含义 (如 Openness: 想象力 vs 务实)。
    *   **用途**：科研、职业规划等。
    *   **权威链接**：直接链接到 Wikipedia 或权威心理学网站。
*   **情绪模型全解**：
    *   展示 16 种情绪的分类表 (A-E 组)。
    *   展示 Valence-Arousal-Dominance 三维排序口诀 (如“羞焦愧怕恶最苦”)。
    *   提供“一句话超级口诀”帮助记忆。

### 2. 增强版角色卡片 (Soul Card Pro)
*   **增加置信度**：在卡片下方标注 "Based on X analysis points" (基于 X 个分析点)，强调数据来源。
*   **权威印章**：在卡片角落增加 "Psychologically Grounded" 的视觉印章。

### 3. 交互优化
*   **Tab 结构调整**：将 `SoulPrint` 分为 `My Soul` (我的灵魂报告) 和 `Library` (心理学图书馆) 两个子 Tab，避免信息过载。

## 技术实现
1.  **数据层**：无需变动，沿用已构建的 `PSYCH_KEYWORDS` 和 `EMOTION_KEYWORDS`。
2.  **UI 层**：
    *   使用 `st.expander` 构建科普模块。
    *   使用 `st.markdown` 渲染富文本解释和链接。
    *   嵌入用户提供的口诀和分类表。

## 执行步骤
1.  **构建 Library 模块**：编写六大模型和情绪模型的详细科普文案 (Markdown)。
2.  **集成 UI**：在 `SoulPrint` 页面底部或侧边栏加入 Library 模块。
3.  **最终验证**：检查所有链接是否有效，文案是否准确引用用户提供的权威描述。
