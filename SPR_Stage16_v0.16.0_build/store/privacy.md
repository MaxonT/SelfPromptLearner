# Privacy Policy (Draft)

SPR does not sell your data.
- Captured prompts are stored locally in your browser and optionally synced to YOUR server.
- If you enable sync, prompts are sent to the server URL you configure.
- Account data can be exported (JSON/CSV) and deleted at any time.

Collected data:
- Prompt text and minimal metadata (site, url, timestamps)
- Device ID (random identifier for sync idempotency)
- User account identifier on your server

Contact: privacy@example.com
