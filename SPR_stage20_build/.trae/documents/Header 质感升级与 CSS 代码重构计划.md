# 1. 重构 Header UI (Luxury Upgrade)

**目标**: 去除"灰灰的"廉价感，打造沉浸式、高通透度的 Luxury 头部。
**方案**:

* 移除 Header 默认背景色，改用透明背景 + 局部磨砂玻璃 (`backdrop-filter`)。

* 移除底部边框，让头部与内容区自然融合。

* 操作区按钮改为半透明悬浮态，Hover 时显示金色光晕。

# 2. 清理冗余 CSS (Maintainability)

**目标**: 精简代码，移除未使用的类和重复定义。
**方案**:

* 删除 `.list-header` 重复定义。

* 合并滚动条样式。

* 统一 `[data-theme="light"]` 的覆盖逻辑，避免分散在各处。

# 3. 增强细节质感

**目标**: 提升交互体验。
**方案**:

* Tab 栏: 增加更平滑的过渡动画，激活态增加微弱的金色外发光。

* 字体: 增强标题的字重对比，使其更具杂志感。
